# Turbo Drive

Simulador de conducción arcade con estética ochentera en el que el jugador debe recoger monedas y sobrevivir al turbo.

## Objetivo del proyecto

El objetivo de este proyecto es practicar desarrollo web, lógica de juego e interacción en tiempo real mediante una experiencia de conducción arcade sencilla y directa.

La propuesta busca capturar una sensación retro: velocidad, reflejos, supervivencia y puntuación, todo dentro de una app ligera orientada a navegador.

## Características principales

- Simulador de conducción arcade.
- Inspiración visual y conceptual en los videojuegos ochenteros.
- Mecánica de recogida de monedas.
- Enfoque de supervivencia y reflejos.
- Proyecto ligero pensado para navegador.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Creación de una experiencia arcade en navegador.
- Gestión de eventos de usuario.
- Lógica de movimiento y obstáculos.
- Sistemas básicos de puntuación o progresión.
- Diseño de una app web con identidad retro.
